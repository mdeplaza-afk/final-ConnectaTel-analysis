# final-ConnectaTel-analysis
# Análisis empresa de telecomunicaciones con segmentación de usuarios.
README – Proyecto de Análisis de Usuarios ConnectaTel

# Descripción del Proyecto
Este proyecto realiza un análisis completo de los usuarios de ConnectaTel, integrando información de perfiles, uso de llamadas y mensajes, y segmentación por edad y nivel de uso. El objetivo es identificar patrones de comportamiento, detectar oportunidades comerciales y generar recomendaciones estratégicas basadas en datos.

# Contenido del Repositorio
-	notebooks/ – Notebooks con el análisis paso a paso.
-	scripts/ – Funciones de limpieza, segmentación y reportes.
-	results/ – Tablas, gráficos y reportes generados.
-	README.md – Documentación del proyecto.

# Objetivos del Análisis
1.	Evaluar la calidad e integridad de los datos.
2.	Integrar datasets de usuarios y uso.
3.	Crear segmentaciones por edad y nivel de uso.
4.	Identificar patrones, outliers y oportunidades comerciales.
5.	Generar recomendaciones basadas en los hallazgos.

# Calidad de los Datos
-	No se encontraron valores nulos significativos.
-	La estructura de los datasets era consistente.
-	No se eliminaron filas ni se modificaron datos originales.
-	Los únicos problemas detectados fueron outliers en variables de segmentación, representando menos del 1%.
	
# Segmentación de Usuarios
## Por Edad
-	Joven
-	Adulto
-	Adulto Mayor

## Por Nivel de Uso
-	Bajo uso
-	Medio uso
- Alto uso

# Hallazgos Principales
-	Los adultos con bajo y medio uso representan más del 50% cada uno.
-	Los usuarios de alto uso generan ingresos significativos.
-	Los usuarios de bajo uso representan una oportunidad de crecimiento.

# Recomendaciones Estratégicas
•	Crear un plan intermedio para usuarios de bajo y medio uso.
•	Ajustar beneficios para incentivar migración a planes premium.
•	Campañas de activación para usuarios de bajo uso.
•	Fidelización para usuarios de alto uso.
•	Comunicación segmentada por edad.
